Beta notice

This help center describes the public v0.3.3 Mac beta. Expect rough edges. Read these constraints before you install:

  • macOS 13+ (Ventura or newer).
  • Apple Silicon builds: cloud OpenRouter STT plus optional local MLX (explicit ~1.5 GB download). Intel builds: cloud-only.
  • Builds are not Developer ID signed or notarized. First launch: right-click golos → Open (Gatekeeper).
  • Updates are manual — download a newer DMG from the release page. There is no automatic update installer in this beta.
  • Windows desktop and mobile are in the pipeline with no announced date. Mac is supported today.
  • Fully local processing requires Apple Silicon with the local MLX model, Format with LLM off, and the optional learning reviewer off. Cloud STT still sends audio when you use OpenRouter.
  • A green ✓ inserted flash means insert events were posted after Accessibility preflight — not a guarantee that every app or text field accepted the text.
  • Intermittent UI or insertion glitches: menu → Export Diagnostics… before restarting. Bundles stay local until you share them and exclude keys, audio, and transcript/prompt/context content.
  • Context-aware citations and path resolution are best-effort from accessible context; they are not guaranteed in every app.

First five minutes

Short path after you have a DMG

  1. Install

    Open the DMG for your architecture, drag golos to Applications, then right-click → Open once.

  2. Permissions

    Grant Microphone, Input Monitoring, and Accessibility to golos.app, then relaunch (especially after Input Monitoring). Set Keyboard → “Press 🌐/fn key to” → Do Nothing.

  3. Connect STT

    Add an OpenRouter key in onboarding for cloud transcription, or on Apple Silicon download the local model from Settings → General when you want on-device STT.

  4. Dictate

    Hold fn, say something, release. Use menu-bar → Test insertion if nothing lands at the cursor.

Raw vs formatted at a glance

Toggle in Settings → General → Format with LLM

Mode What runs Speed Privacy note
Raw STT only; no LLM cleanup Fastest insert Skips formatter and context sharing. Cloud STT still sends audio if OpenRouter is selected.
Formatted (default) STT + optional LLM polish, lists, corrections, context Typically a short extra wait Transcript and allowed context can leave the Mac when formatting is on.

Where state lives

Config, dictionary, corrections, history, and optional recordings live under ~/.golos/ on your Mac. You can read, move, or delete them. golos is free and open-source under MIT — cloud model usage is billed by the provider you choose, not by a golos subscription.
